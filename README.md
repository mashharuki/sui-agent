# sui-agent

ElizaOS + Sui Pluginを試す検証用のリポジトリ

## CLIのインストール

```bash
bun i -g @elizaos/cli
```

以下のコマンドを実行してバージョンが出力されればOK!

```bash
elizaos --version
```

## テンプレアプリ生成

```bash
elizaos create
```

## 動かし方

`sui-agent`ディレクトリ配下で実行すること

```bash
bun install
```

起動

```bash
bun run dev
```
