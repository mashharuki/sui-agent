# Project Summary

## Purpose

This is a repository for verifying the ElizaOS + Sui Plugin. It appears to be a starter project or template for building agents with the ElizaOS framework.

## Tech Stack

- **Runtime/Package Manager:** Bun
- **Language:** TypeScript
- **Frontend:** React, Vite
- **Styling:** Tailwind CSS
- **Framework:** ElizaOS
- **Testing:** Cypress
- **Linting/Formatting:** Prettier, Biome
- **Schema Validation:** Zod
